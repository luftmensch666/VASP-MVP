## 4. 配置文件 (`config/`)

存放各种全局配置和国际化文本：
- **`defaults.json`**: 包含 VASP 路径、默认 MPI rank 等。
- **`potcar_map.json`**: 包含元素与赝势的映射。
- **`vaspkit_options.json`**: 定义了与 VASPKIT 交互可用的相关选项。
- **`i18n/`**: 包含 `zh.json` 和 `en.json`，用于界面文本的多语言支持。
